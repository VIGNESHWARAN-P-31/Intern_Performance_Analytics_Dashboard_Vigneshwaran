# Intern_Performance_Analytics_Dashboard

This project focuses on analyzing intern performance using data analytics techniques. The objective is to understand intern productivity, attendance, task completion, and mentor evaluation.

## Sprint 1 – Data Collection & Preparation
- Defined dataset structure
- Created simulated dataset with 100 records
- Performed data cleaning
- Preprocessed data using Python (Pandas)
- Prepared dataset for analysis

## Sprint 2 – Exploratory Data Analysis (EDA)
- Performed statistical analysis
- Identified trends and patterns
- Created visualizations using Matplotlib and Seaborn
- Analyzed relationships between attendance, performance, and mentor ratings

## Sprint 3 – KPI Dashboard Development
- Defined Key Performance Indicators (KPIs)
- Developed interactive dashboard using Power BI
- Visualized intern performance metrics
- Created dynamic and insightful reports
